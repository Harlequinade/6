PestPatrol Corporate 4.4.3
Pest Patrol - мощный антивирус, программа предназначена для поиска и удаления различных вредоносных программ, которые обычно выпадают из поля зрения традиционных антивирусов: "бомб", "троянцев", средств для организации DDoS-атак и т. д. Всего более 17 тыс. наименований. В эту категорию попадают и spyware-модули.
Установка:
Для нормальной работы программы необходимо, чтобы в системе был установлен Microsoft .NET Framework, для его инсталляции запустите dotnetfx.exe
Для установки запустите setup.exe
