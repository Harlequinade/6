  • Versión 1.02

  • Ahora LookUp (Buscar) puede trabajar con lista de direcciones

  • Ahora el programa puede sincronizar la hora con los servidores horarios en modo automático
    si usted ejecuta el programa con los siguientes parámetros :
    >ip_tools.exe /SetTime TryCount Host1 [Host2 [Host3 [...]]
    donde :
       TryCount - cuenta los intentos de conexión con los servidores horarios (0..255)
       Host1 .. HostN - indica las direcciones de los servidores horarios
    el programa intenta conectar con el Host1, y si la conexión es satisfactoria el programa obtiene la hora del servidor horario, configura esta hora (+GMT) en la máquina local y culmina. Si el programa no consigue conectarse al Host1 entonces intenta conectarse al Host2, ... Si no consigue conectarse ni siquiera con el último servidor (HostN) el programa intenta nuevamente conectarse al Host1. El programa repite el proceso la cantidad de veces establecida.
    Ejemplo :> ip_tools.exe /SetTime 2 xa.xa.net 192.43.244.18
