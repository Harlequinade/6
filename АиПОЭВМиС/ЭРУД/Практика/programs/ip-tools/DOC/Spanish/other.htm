Requisitos de Sistema:

  • Microsoft Windows 95/98/ME/NT 4.0/2000 o Windows XP (recomendado Windows 2000)
  • Protocolo Microsoft TCP/IP

Cómo Instalar / Actualizar / Desinstalar IP-Tools

Cómo instalar IP-Tools Ejecute ip-tools.exe, seleccione un directorio de destino, y presione el botón [Install].
Nota: Si usted ya tiene una versión de IP-Tools, el programa sobrescribirá todos los archivos existentes (puede perder alguna de sus configuraciones). Para mantener las configuraciones utilice el botón [Update] (Actualizar).
Cómo actualizar IP-Tools

Ejecute ip-tools.exe y presione el botón [Update] (Actualizar). El programa sobrescribirá los módulos ejecutables, la documentación, etc, pero conservará las configuraciones.
Cómo desinstalar IP-Tools    Existen tres formas diferentes de desinstalar IP-Tools de su computador:
Variante 1 : Ejecute uninstall.exe, que se encuentra en el directorio IP-Tools (o desde el grupo de programas IP-Tools) y haga clic en el botón "Start".
Variante 2 : Abra el Panel de Control de Windows, haga un doble clic en el incono de "Agregar/Quitar Programas", seleccione "IP-Tools" de la lista y haga clic en el botón "Agregar/Quitar".
Variante 3 : Borre todos los archivos de su directorio IP-Tools, borre el directorio, remueva el icono de IP-Tools de su escritorio, y remueva el grupo de programas IP-Tools.


Problemas Conocidos:

Nota: Utilizando una versión de IP-Tools con un crack, o ejecutando el programa con una licencia creada ilegalmente hará que el programa se vuelva inestable y se cuelgue frecuentemente.

Problema   Recomendación
Aparece un mensaje de error como este: "SNMP extension agent library is not available"  

Este mensaje aparece en algunos sistemas, cuando usted inicia el utilitario IP-Monitor. Significa que el programa no puede cargar el archivo inetmib1.dll. Normalmente este DLL debe estar presente en su directorio Windows (Windows\System32 en Windows NT y Windows 2000). Si los archivos inetmib1.dll y snmpapi.dll no se encuentran presentes, extráigalos del CD de instalación de Windows, o cópielo de otro computador.
