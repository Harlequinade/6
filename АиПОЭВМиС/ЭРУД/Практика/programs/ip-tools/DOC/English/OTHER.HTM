System Requirements:

  • Microsoft Windows 95/98/ME/NT 4.0/2000 or Windows XP (Windows 2000 recommended)
  • Microsoft TCP/IP protocol

How To Install/Upgrade/Uninstall IP-Tools

How to install IP-Tools Run ip-tools.exe, select a destination folder, and press the [Install] button.
Note: If you already have installed a version of IP-Tools, the program will overwrite all existing files (you may lose some of your settings). To retain settings use the [Update] button.
How to upgrade IP-Tools

Run ip-tools.exe and press the [Update] button. The program will overwrite executable modules, documentation, etc, but keep all your previous settings.
How to uninstall IP-Tools    There are three different ways to uninstall IP-Tools from your computer:
Variant 1 : Run uninstall.exe found in the IP-Tools directory (or from the IP-Tools program group) and click the "Start" button.
Variant 2 : Open the Windows Control Panel, double click the "Add/Remove Programs" icon, select "IP-Tools" from the list and click the "Add/Remove" button.
Variant 3 : Delete all files from your IP-Tools directory, delete the directory, remove the IP-Tools icon from the desktop, remove the IP-Tools program group.


Known Problems:

Note: Using a cracked version of IP-Tools, or using IP-Tools with an illegally cteated licence key makes the program unstable and frequently crashing.

Problem   Recommendation
An error message is displayed, "SNMP extension agent library is not available"  

This message appears on some systems when you start IP-Monitor utility. This means the program cannot load the inetmib1.dll. Normally this DLL must be present in your Windows directory (Windows\System32 in Windows NT and Windows 2000). If the files inetmib1.dll and snmpapi.dll are not present, extract them from the Windows installation CD or copy them from another computer.
