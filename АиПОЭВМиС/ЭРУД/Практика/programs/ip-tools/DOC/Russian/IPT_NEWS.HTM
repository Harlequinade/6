      Полностью переделан "Options" диалог:
  • добавлено 7 новый страниц настроек : HostMonitor, LogSettings, HTML Colors, WhoIs, Lookup, etc.

  • теперь строка состояния отображает иконки для работающих утилит. Кликнув мышкой по иконке можно сделать утилиту активной (видимой).
  • все утилиты которые могут работать со списком или с диапазоном адресов отображают "progress bar" и расчетное время необходимое для окончания работы.

  • кнопки для каждой утилиты

  • кнопку для выбора утилиты которая была активной последний раз

  • новый пункт меню "Save as Text" (hot key: Ctrl+S). Работает для всех утилит кроме IP-Monitor и HostMonitor.
  • новый пункт меню "Save as HTML" (hot key: Ctrl+H). Работает для Local Info, Connections Monitor, NBScanner, PortScanner, HostMonitor.

  • пункт "Show/Hide Toolbar"
  • пункт "Show/Hide Tabs"

      Local Info
  • отображает информацию о сетевых интерфейсах (получает информацию двумя разными методами: NetBIOS и SNMP)
  • отображает статистику по протоколам IP, ICMP, UDP, TCP
  • отображает информацию о модемах (пока только на Windows NT/2000)
  • выдает детальную информацию о процессоре
  • показывает таблицу маршрутизации
  • всю информацию можно записать в текстовый или HTML файл
      Connections monitor
  • новая опция "Display Host names". Теперь IP-Tools может отображать имена хостов вместо IP адресов
  • добален новый пункт в popup меню: "Save as HTML..". Теперь можно создавать HTML отчет.

  • теперь сканнер раз в 10 быстрей, может сканировать много машин одновременно
  • новая опция "Detect OS" (работает только если вы работаете под Windows NT/2000/XP).
  • новая опция "Ping : skip address if host no answer"

  • новая опция "Sound when resource found"
  • можно использовать команды из popup меню: "Select All (Ctrl+A)" и "Copy Ctrl+C" для копирования текста в clipboard
      Port Scanner
  • новая опция "GetHostName : skip address if not resolved"
  • новая опция "Ping : skip address if host no answer"
  • новая опция: "Show port's descriptions"
  • новая команда в popup меню "Save as HTML..". Можно создавать HTML отчеты
      Ping Scanner
  • теперь сканнер выдает реальные ответы, а не только "reply" или "timeout".
      Trace
  • новая опция "stop trace if any device does not respond"
      WhoIs
  • поддерживает новые .BIZ и .INFO домены
  • теперь Вы можете создать собственный список WhoIs серверов. См. "WhoIs" закладку в Options диалоге.
      Lookup - серьезно усовершенствован. Можно устанавливать:
  • DNS сервер
  • порт и протокол (TCP, UDP)
  • таймаут и количество попыток получить ответ от сервера
  • тип запроса: A, ANY, CNAME, HINFO, MX, NS, PTR, SOA, или TXT
  • включить или отключить рекурсию
  • режим вывода информации: Full, Medium, или Compact
      Telnet
  • новая команда "Paste" в popup меню терминала
  • новая команда "Change font..." в popup меню терминала
      HostMonitor
  • теперь монитор в 10-20 раз быстрее потому что тестирует несколько хостов одновременно
  • новая опция "Style : Report / List / Icons / Small icons"
  • новая опция "Show grid lines : Yes / No"
  • новая опция "Refresh status after edit host's properties : Always / Ask / Never"
  • новая опция "Start/Stop mode : start after load / stop after load / restore last status"
  • теперь IP-Tools может менять имя лог файла каждый день/месяц/год. Вы можете задавать макросы в имени файла (%dd%, %mm%, %yy%, %YYYY%, etc). Например: если определить имя лог файла как "c:\%ddmmyy%.log" то программа будет подставлять вместо имени текущую дату, тоесть менять имя файла каждый день; если "f:\%YYYY%.log - то программа будет менять лог файл раз в год.
  • теперь IP-Tools может записывать в log файл информацию о всех тестах, даже если хост не изменил состояния. Удобно если Вы например хотите иметь статистику по времени доступа к хосту.
  • новый пункт в popup меню: "Create HTML report".
  • новая опция "Log Format". Теперь IP-Tools может вести лог не только в текстовом но и в HTML формате.
  • но если Вам все таки нужен намного более мощный, быстрый и удобный монитор с кучей тестов, то Вам сюда HostMonitor
      ScanList
  • теперь можно использовать отдельные списки для каждой утилиты или как раньше использовать один список для всех утилит
  • теперь IP-Tools хранит ScanList в ini файле и Вы можете создать несколько профайлов или в любой момент выбрать другой набор настроек

    В команде можно задавать макроподстановки :
       %DateTime% - IP-Tools подставит дату и время
       %HostName% - IP-Tools подставит имя хоста
    В каталоге "SAMPLES\" есть примеры :
      - "message.exe" - программа message которая просто выводит диалоговое окно с текстом который передается через командную строчку
      - "sendmail.exe" - программа которая может отсылать письма без участия человека при этом все нужные параметры можно передать через командную строчку
    Например можно задать такую команду "samples\sendmail.exe To=admin@firm.com "Subject=%datetime% host %hostname% is Dead" StartAfterLoad"
    А поскольку по e-mail можно отправлять сообщения даже на пейджер то IP-Tools может сообщить админу о падении сервера даже на пляже (чему он несомненно обрадуется ;-)

    Но если Вам нужен навороченный монитор для контроля вашей сети Вы можете попробовать Advanced Host Monitor.
